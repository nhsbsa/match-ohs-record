MVP 1 - SBUC 5 - Unhappy Path #2 'S044 partial address, no OHS record, create OHS record without address' user flow

View the prototype "Unhappy Path #2"

How to use prototype

  1. Click to search OHS records
  2. Click to 'Create new OHS record'
  3. Select 'Without an address' then click 'Create new OHS record'
  4. Click 'OHS reference' of record to go to OHS and make a note (not included) - End of task 1
  5. Click View H002 and copy address to notepad
  6. Click to 'Seach OHS records'
  7. Click '01234567' OHS reference to open OHS to update the record and add address
  8. Close OHS browser tab and return to case management tab
  9. ...

Design summary

  1. The SED S044 comes in with an incomplete address i.e. only with Town and Country data items available in the address section as these are the only ones mandatory.
  2. SED S044 also has a SED S041 as an attachment
  3. User determines the S041 is evidence enough and issues an S045 entitlement

User journey key points

  • User is managing a new case where its S044 have an overseas address
  • User goes to find a matched OHS record which has the same overseas address
  • User issues entitlement but ends up with a 'Pending check' due to indesser checks
  • User manages the entitlement by rejecting the entitlement
