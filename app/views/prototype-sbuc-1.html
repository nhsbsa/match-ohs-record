User flow prototypes

Posted worker prototypes

Aims:

  • Enable the user to easily view HMRC X007 responses by posting response in case management screen
  • Enable a user to make a decision on entitlement early in the user flow which changes the onward flow basedon their decision

What's new:

  • HMRC are now represented in the timeline to increase the visibility of their responses
  • HMRC X007 responses and reason is now displayed in the case management screen to help users quickly make a decision on next steps
  • Decision on entitlement is now appearing very early in the user flow to help tailor the onward experience based on their decision
